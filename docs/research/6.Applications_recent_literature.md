# Recent literature candidates for `tex/6.Applications.tex`

Updated: 2026-04-26

This note collects recent papers that can strengthen Section 6, `Potential Applications`. I checked the current section and `tex/zotero.bib`. Most candidates below are not yet in the bibliography. The current section has four application directions: online social networks, recommender systems, knowledge management, and biology. The strongest recent evidence is in recommender systems and biomedical/molecular applications; social-network-specific graph prompting is still relatively sparse.

## Suggested Section-Level Revision Plan

1. Keep the four current application headings, but reduce speculative wording.
2. Recommender systems should be the most concrete application paragraph. Add `GraphPro`, `GPT4Rec`, `PT4Rec`, and optionally `CGPT`.
3. Online social networks should be reframed as dynamic interaction and security scenarios. Add `DyGPrompt`; optionally cite `UniGAD` for anomaly detection, while noting it is not itself a graph-prompt method.
4. Knowledge management should distinguish direct KG prompt tuning from KG-grounded LLM prompting. Add `KnowGPT` and `GNN-RAG`; add `APKGC` only if discussing KG completion.
5. Biology should expand from general molecule-language models to concrete biomedical graph prompting. Add `MOAT`, `DDIPrompt`, `Prompt-DDG`, and optionally `MolFinePrompt`.
