## 🔴 严重问题详情

### 1. Self-Pro — 下游任务覆盖范围与文档不符

**文档描述**（`docs/GNN/Self-Pro/方法介绍.md` 第 88-94 行）：
- 明确写出"第三类，节点分类"和"第二类，链路预测"，且两者共用同一个 `L_dow` 框架
- 链路预测损失：$L_{dow}= -\sum_{(v,a,b)\in T} \log \frac{\exp(t_v'^T t_a'/\tau)}{\exp(t_v'^T t_a'/\tau)+\exp(t_v'^T t_b'/\tau)}$

**表格标记**（`tex/table/graph_prompt_summary.tex`）：
- node ✓, edge ✗, graph ✗

**问题：** 链路预测是文档明确描述的下游任务之一，表格却没有标记 edge 支持。

---

### 2. ProNoG — 预训练任务覆盖范围与文档不符

**文档描述**（`docs/GNN/ProNoG/方法介绍.md` 第 165-176 行）：
- 明确选择 **non-homophily pre-training**，主要用 **GraphCL**（属于 node-level 对比学习）
- 论文实验以此为默认设置

**表格标记**：
- node ✓, edge ✓, graph ✓（三种预训练任务全标注了）

**问题：** GraphCL 是 node-level 预训练，文档并没有说它同时覆盖 edge 或 graph 级别。表格可能多标了。

---

### 3. UniPrompt — Prompt tuning 机制与文档不符

**文档描述**（`docs/GNN/UniPrompt/方法介绍.md` 第 67-70 行）：
- 明确写出端到端可学习的目标函数
- 优化的是 $\Psi$（边权重矩阵），通过梯度下降更新
- 损失函数：$\min_{\phi,\Psi}\frac{1}{|V_L|}\sum_{v_i\in V_L} \ell_D(\cdots)$

**表格标记**：
- Preset ✓, Learnable ✗（即标记为"不可学习"的预设机制）

**问题：** UniPrompt 是可学习端到端训练的，表格却标记为 Preset。这是方向性错误。

## 🟡 中等问题详情

### 4. SUPT — 激活函数 σ(α) 缺失

**文档公式：**
$p_i \leftarrow \sum_j \sigma(\alpha)_{i,j}\textbf{b}_j$，其中 σ 是 softmax 或 rank-selection 激活函数

**表格公式：**
$\textbf{p}_i \leftarrow \sum_j \alpha_{i,j}\textbf{b}_j$（没有 σ(α)）

### 5. SUPT — 聚合跳数 m 缺失

**文档公式：**
$\alpha = (\tilde{D}^{-\frac{1}{2}}\tilde{A}\tilde{D}^{-\frac{1}{2}})^m \left( X \oplus \sum_{j=1}^{k} b_j \right)W$

**表格公式：**
缺失 m 参数

### 6. PSP — GNN 融合公式在表格中缺失

**文档公式：**
$G' \leftarrow (V \cup \mathcal{P}, E \cup \textbf{W})$（仅展示了图扩展）
$\textbf{P} \leftarrow \mathrm{GNN}([\textbf{X},\textbf{X}_{P}],[\textbf{A},\textbf{W}])$（关键融合公式）

**表格公式：**
仅展示了第一行

### 7. IGAP — 公式极度压缩

**文档公式：**
- $\tilde{\textbf{x}}_{i} \leftarrow \textbf{x}_{i} + \sum_{s} \alpha_{is} \textbf{p}_{s}$

## arxiv 原文核查结论（2026-04-26）

### Self-Pro — ✅ 确认 edge 下游任务存在

**论文：** Self-Pro: A Self-Prompt and Tuning Framework for Graph Neural Networks
**来源：** arXiv:2310.10362v3, ECML-PKDD 2024

**核查结果：**
- **节点分类**：✓ 明确支持
- **链路预测（Link Prediction）**：✓ Table 6 明确用 AUC 和 AP 指标评估链路预测
- **图分类**：论文中**未提及**

**表格现状：** node ✓, edge ✗, graph ✗

**应修改为：** node ✓, edge ✓, graph ✗

---

### ProNoG — ✅ 确认预训练任务范围

**论文：** Non-Homophilic Graph Pre-Training and Prompt Learning
**来源：** arXiv:2408.12594v6

**核查结果：**
- **预训练任务**：以 **GraphCL** 为主（非同配预训练），GraphCL 是 node-level 对比学习方法
- **下游任务**：明确写 "node classification and graph classification"，**链路预测未提及**
- **Prompt 可学习性**：端到端可学习（condition-net 生成节点专属 prompt）

**表格现状：** pre-training 标记为 node ✓, edge ✓, graph ✓（三种全标注了）

**应修改为：** pre-training 应为 node ✓ 为主（GraphCL 是 node-level），edge ✗, graph ✗。表格目前多标了。

---

### UniPrompt ("One Prompt Fits All") — ✅ 确认 prompt 为可学习

**论文：** One Prompt Fits All: Universal Graph Adaptation for Pretrained Models
**来源：** arXiv:2509.22416v2, NeurIPS 2025

**核查结果：**
- **下游任务**：仅涉及 **node classification**（同域和跨域 few-shot 节点分类），链路预测和图分类**均未提及**
- **预训练任务**：使用 DGI、GRACE、GraphMAE、FUG 等多种预训练模型
- **Prompt 可学习性**：**端到端可学习**。优化目标为 $\min_{\phi,\Psi}\frac{1}{|V_L|}\sum_{v_i\in V_L} \ell_D(\cdots)$，其中 $\Psi=\{w_{ij}\}$ 为可学习的边权重参数，$\phi$ 为分类器参数，与预训练编码器 $f_\theta$ 联合优化
- **这不是 Preset 方法**，而是可学习端到端训练的

**表格现状：** Preset ✓, Learnable ✗

**应修改为：** Preset ✗, Learnable ✓
