\section{Conclusion}\label{sec:con}
This survey reviews graph prompt learning as an emerging adaptation paradigm for more transferable and reusable graph learning systems. We organize the literature around transfer across tasks, domains, and modalities, and relate graph prompting to graph pre-training, graph foundation models, multi-modal graph learning, and practical applications. To support systematic comparison, we summarize graph prompts through a unified framework covering prompt tokens, token structures, and insertion patterns.

Beyond method taxonomy, we review application evidence in online social networks, recommender systems, knowledge management, and biomedical graph learning, and discuss supporting resources such as ProG. The field has made clear progress, but it remains far from mature. Key open problems include reusable graph foundation models, reliable transfer protocols, evaluation beyond downstream accuracy, stronger theory, and more interpretable prompt designs. We hope this survey provides a coherent reference point for future work on graph prompting and general-purpose graph learning.
