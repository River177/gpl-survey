\IEEEtitleabstractindextext{
\begin{abstract}
\justify{通用人工智能（Artificial General Intelligence, AGI）已在诸多领域引发深刻变革；然而，其与图数据（这一互联世界中的核心数据形态）的融合仍处于起步阶段。本文针对 AGI 语境下新兴的“图提示（graph prompt）”研究方向，系统梳理了利用图数据支撑 AGI 应用的关键挑战与机遇。尽管 AGI 在自然语言处理与计算机视觉等领域取得了显著进展，但面向图数据的研究相对不足。本文批判性地回顾了 AGI 处理图数据的现状，重点讨论图任务在跨模态、跨领域与跨任务场景下面临的独特困难。我们首次提出用于理解图提示学习的统一框架，明确了图提示中的提示 token、token 结构以及插入模式等核心要素。进一步地，我们分析了图提示的内在属性，探讨其灵活性、表达能力及其与现有图模型之间的关系。我们给出覆盖 100+ 相关工作的系统化分类体系，并将其与节点级、边级与图级等不同预训练目标进行对齐。此外，我们还发布了 Python 库 ProG 以及配套网站，以支持并推动图提示研究。最后，本文总结当前挑战并展望未来方向，为 AGI 时代的图提示研究提供路线图。通过上述全面分析，我们希望促进 AGI 在图数据上的进一步探索与落地应用，凸显其重塑 AGI 相关领域及更广泛场景的潜力。
ProG 与相关网站分别可通过 \url{https://github.com/WxxShirley/Awesome-Graph-Prompt} 与 \url{https://github.com/sheldonresearch/ProG} 访问。}
\end{abstract}
