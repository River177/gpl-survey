\IEEEtitleabstractindextext{
\begin{abstract}
\justify{通用人工智能（AGI）加速了跨领域基础模型的研究，然而图数据在这一范式中仍不如文本和视觉数据受到广泛关注。本综述从跨模态、跨域和跨任务泛化的视角审视图提示学习，并基于提示标记、标记结构和插入模式建立了统一框架。本综述进一步探讨了图预训练与提示的关系，讨论了提示如何将下游任务与代理目标对齐，并总结了具有代表性的提示调优策略。该分类体系涵盖了过去数年在一百余项代表性研究，涵盖预训练策略、提示设计、多模态提示和图域自适应等方向。我们还总结了应用场景和支持资源，包括ProG以及精心整理的论文、基准测试和代码库。最后，本综述讨论了当前挑战和未来方向，为图表示学习及相关面向通用人工智能的应用中图提示的未来发展提供了展望。相关资源请访问 \url{https://github.com/WxxShirley/Awesome-Graph-Prompt} 和 \url{https://github.com/sheldonresearch/ProG}。
}

\end{abstract}
