\section{引言}\label{sec:intro}


在人工通用智能（Artificial General Intelligence, AGI）快速演进的时代背景下，基于AGI技术涌现出诸多令人瞩目的应用，例如自然语言处理（NLP）领域的ChatGPT以及计算机视觉（CV）领域的Midjourney。AGI显著改善了我们的生活方式，使工作更高效，并将人们从重复性劳动中解放出来，从而能够将精力投入到更具创造性的任务中。然而，与其在NLP \cite{
devlin2019bert,
brown2020language, liu2023pretrain} 与CV领域 \cite{
% radford2021learning,
% li2022blip,
% zhu2023minigpt4,
wang2023chatvideo,
zhang2023videollama} 所取得的巨大成功相比，面向图数据的AGI应用仍处于相对早期阶段。在日益互联的世界中，理解图结构并从中提取有价值的洞见至关重要。这使得“将AGI应用于图数据”成为学术界与工业界共同关注的前沿方向 \cite{liu2023graph, zhang2023large, yang2023datacentric}，并有潜力重塑药物设计 \cite{rong2020selfsupervised, qian2023can}、电池研发 \cite{wang2023scientific} 等领域。

然而，实现这一愿景并非易事。图\ref{fig:AGIProblems}刻画了近期人工通用智能研究所面对的整体图景：至少存在三个基础性技术问题——如何使模型能够泛化到不同模态、不同领域以及不同任务？在NLP与CV领域，已经出现了多种商业化模型能够在这些模态间理解并转换信息 \cite{
devlin2019bert,

brown2020language}。例如，BERT \cite{devlin2019bert} 与GPT-3 \cite{brown2020language} 展示了在涉及文本与视觉信息的任务上进行有效推理的能力。然而，在图数据语境下，多模态信息的协同与统一仍在很大程度上处于未知领域 \cite{li2023graphadapter}。

针对跨领域问题，迁移学习在许多场景中已被证明行之有效，使模型能够将某一领域中从图像或文本学到的知识迁移到另一领域。然而，不同图领域之间的知识迁移更加困难：一方面语义空间往往不对齐 \cite{zhu2023graphcontrol}，另一方面结构模式也可能差异显著 \cite{ zhao2023graphglow}，因此图领域自适应仍是一个前沿且未被充分解决的AGI难题。当前，多数图迁移学习研究更多聚焦于第三个问题：如何在同一图领域内，利用预训练得到的图知识来完成不同图任务（如节点分类、链接预测、图分类等）\cite{sun2022gppt, liu2023graphprompt, sun2023all, fang2023universal, zhu2023sglpt, hu2020gptgnn, shirkavand2023deep, ge2023enhancing}。然而，与NLP和CV的巨大成功相比，同一图领域内的任务迁移仍较为初级，成功的工业级应用实例也相对稀少。尽管AGI在图像、
% \cite{
% % radford2021learning,

文本 \cite{robinson2023leveraging, devlin2019bert, brown2020language} 与视频 \cite{wang2023chatvideo, zhang2023videollama} 等线性数据上取得了显著进展，图数据场景中的基础问题仍远未被充分探索。除上述三类基础问题外，AGI还引发了多种社会争议。例如，训练大型基础模型需要消耗极其可观的能源，并可能产生非预期的反事实结果 \cite{liu2022ptuning, schick2021it}。这些担忧促使AI社区逐渐形成共识：应更高效地从大模型中提取其所蕴含的有用知识，尽可能减少在不同下游任务上反复微调的需求 \cite{gao2021making, lester2021power}。这一共识不仅有望缓解环境影响，也为AGI时代模型效率与适应性问题提供了切实可行的解决路径。



在近期AGI技术的核心位置，提示学习（prompt learning）展现出解决上述问题的巨大潜力，并在NLP与CV应用中取得了显著成功 \cite{qin2021learning, tsimpoukelli2021multimodal, liu2023pretrain}。提示学习的本质是设计信息充分的提示（prompt），以操控输入数据并驱动预训练基础模型完成任务。图\ref{fig:PromptExample}给出了一个文本格式提示用于预训练语言模型直接完成下游推理任务的示例。通过将下游任务重表述为预训练任务，该方法能够避免对模型进行大规模调参，从而更高效地提取模型已保存的知识 \cite{brown2020language, jiang2020how}。由于其在数据操控、任务重构以及关键信息提取方面的强大能力，提示学习有望以一种统一的方式应对前述跨模态、跨领域与跨任务挑战。与大模型参数规模相比，提示通常更轻量，能够减少对大模型反复微调带来的高昂算力开销，从而更高效地获得可迁移知识 \cite{lester2021power, shin2020autoprompt}。从直觉上看，文本与图像可视作更一般的图数据结构的特殊情形：例如，句子可被视为一条图路径，其中词为节点；图像可被视为网格图，每个像素对应一个节点。这一洞见启发我们探索将文本领域中成功的提示技术迁移到图学习领域，以应对类似的挑战与诉求。

近期，一些研究者开始将提示学习引入图数据 \cite{sun2022gppt, liu2023graphprompt, sun2023all, fang2023universal, zhu2023sglpt, ma2023hetgpt, guo2023datacentric, chen2023ultradp, gong2023prompt}。然而，后续研究发现，图提示与NLP中的提示存在显著差异 \cite{sun2023all}。首先，相较于语言提示，图提示的设计要复杂得多。经典语言提示通常由预定义短语或附加到输入文本后的可学习向量构成 \cite{brown2020language, gao2021making}，其主要关注点在于提示“内容”。但对于图提示而言，我们甚至不清楚它应当“长什么样”：图提示不仅包含提示“内容”，还包含一个未被明确定义的结构设计问题，即如何组织这些提示token，并将它们无缝整合到原始图中。其次，将下游图任务与预训练任务对齐，比语言任务更为困难 \cite{liu2023graphprompt, sun2023all}。例如，语言模型常见的预训练目标是预测被掩码的词 \cite{devlin2019bert}，因此诸如下游问答、情感分类等任务可较容易被重表述为词级任务 \cite{liu2023pretrain}。不同于NLP中预训练任务之间往往共享较大的任务子空间，图任务跨越节点级 \cite{grover2016node2vec}、边级 \cite{zhang2018link} 与图级目标 \cite{sun2020infograph, sun2021heterogeneous}，使得预训练“伪任务”（pretext）更难直接适配。第三，相较于NLP中通常由可读短语构成的提示，图提示往往对非专业读者缺乏直观可解释性。图提示在图模型中扮演的基础角色及其作用机制仍不够清晰，缺乏系统的理论分析；同时，对于图提示质量也缺少明确的评估标准。此外，关于图提示仍存在大量亟待厘清的问题：这些图提示究竟有多有效？在参数复杂度与训练负担方面效率如何？它们对原始图数据的操控能力有多强、灵活性如何？面对这些复杂问题，迫切需要更深入地挖掘图提示在AGI中的潜力，从而推动我们对这一数据科学前沿方向形成更深刻的理解。



尽管近期已有研究尝试探索图提示，但目前仍缺乏一致的框架或清晰的研究路线。这些工作在视角、方法与目标任务上差异较大，导致图提示研究呈现碎片化格局，进而对该领域的系统化发展形成明显阻碍。因此，有必要以统一框架对该方向的最新进展进行全景式梳理、分析与综合。基于此，我们撰写本综述，旨在呈现现有图提示工作如何尝试解决前述迈向AGI的三个基础问题。进一步地，我们也希望通过回答如下更细粒度的研究问题（Research Questions, RQs）来推动该领域发展：

\begin{itemize}
    \item \textbf{RQ1：如何在统一框架下理解差异显著的既有工作？} 我们的核心目标是理解图提示领域中采用的多样化方法，并将这些不同的思路与技术整合为一个连贯的统一框架。该框架将帮助我们系统把握现有研究，并为未来工作提供坚实基础。
   \item  \textbf{RQ2：为何使用提示？图提示的本质是什么？}
  在本部分中，我们将探究提示的重要性与图提示的基本属性：图提示在图问题中究竟发挥何种作用？它们如何融入复杂图结构，并如何帮助我们实现“使AI系统能够有效处理图数据”这一更宏观目标？这些问题凸显了图提示在面向图信息的AI未来发展中的关键意义。


   \item  \textbf{RQ3：如何设计图提示？} 设计高质量图提示是一项复杂工作。我们将深入讨论图提示的技术细节：图提示在形式上是什么样的、如何实现下游任务与预训练任务的对齐、以及图提示如何被学习得到。围绕这些关键问题的讨论将帮助研究者更好地构建能够契合图数据复杂性的有效提示。

   % \item  \textbf{RQ4: How to Evaluate the Quality of Graph Prompts?}
   % When it comes to graph prompts, it's not as straightforward as evaluating text prompts, which are more accessible to people who aren't experts.    \item  \textbf{RQ4：如何将图提示部署到真实世界应用中？} 目前仍缺少易用的图提示构建工具包，图提示可落地的应用场景也尚未被充分探索。本研究问题聚焦于如何通过易于扩展的编程工具，使图提示在真实世界中更具可用性与可落地性。


   \item  \textbf{RQ5：图提示研究面临哪些挑战与未来方向？} 该问题引导我们审视当前所面临的挑战与可能的发展路径。通过回答这些关键问题，我们希望为持续推进图提示研究提供一份路线图。


\end{itemize}



针对研究问题RQ1，我们提出一个统一框架以分析图提示学习相关工作。该框架将图提示概念拆解为提示token、token结构与插入模式。该高层视角具备清晰性与完整性，能够为读者提供对这一新兴领域的结构化理解。据我们所知，本综述是首次在统一框架下整合并阐释图提示学习中的这些关键要素。



针对研究问题RQ2，我们从灵活性（flexibility）与表达性（expressiveness）的角度，探讨提示与既有图模型之间的关联，并提出一种新颖且富有启发性的视角以揭示图提示的本质。不同于NLP领域多数提示学习综述 \cite{liu2023pretrain} 将提示视为“弥合预训练任务与下游任务差距”的技巧，我们指出图提示与图模型在更深层面上存在内在联系。该视角能够解释提示学习为何在图领域具备潜力，以及其与传统微调方法 \cite{hu2020strategies} 的关键差异。据我们了解，这是首次尝试从这一角度对图提示进行系统阐释。



针对研究问题RQ3，我们构建了一个涵盖100余篇相关工作的综合性分类体系。该分类体系从节点级、边级与图级任务角度对现有工作进行剖析，并将其与更广泛的预训练任务背景对应起来，从而帮助读者更清晰地理解在“预训练—提示”整体流程中提示机制的作用机理。

针对研究问题RQ4，我们开发了统一的Python图提示库ProG（prompt graph）\footnote{\url{https://github.com/sheldonresearch/ProG}}，以支持图提示研究与应用落地。此外，我们还建立了一个网站\footnote{\url{https://github.com/WxxShirley/Awesome-Graph-Prompt}}，作为最新图提示研究成果的持续汇聚平台。该平台收录并维护研究论文、基准数据集以及可复现的代码实现。通过构建这一可访问的生态，我们希望更有效地赋能研究者与实践者，推动该新兴领域的进一步发展。

除上述贡献外，本综述还进一步介绍潜在应用、分析当前挑战并讨论未来方向，从而为该充满活力且持续演进的研究方向提供一份较为完整的发展路线图（RQ5）。我们的主要贡献总结如下：

\begin{itemize}
    \item \textbf{支撑全面分析。} 我们提出用于分析图提示学习工作的统一框架，从提示token、token结构与插入模式三个维度提供系统视角。
    \item \textbf{关于提示—模型关系的新视角。} 我们对图提示的本质给出新的洞见。不同于将提示仅视为“弥合下游任务与预训练任务差距”的技巧，我们从图模型的灵活性与表达性出发，更深入地讨论提示与既有图模型之间的内在关联。
    \item \textbf{图提示研究的系统分类体系。} 我们系统梳理了该领域百余篇近期工作。该分类体系不仅组织了相关研究，也帮助读者理解在“预训练与提示”整体工作流中提示机制如何发挥作用。
    % \item \textbf{Quantitative and Qualitative Insights. } We conduct extensive experiments to evaluate different graph prompts and propose a new metric to evaluate how well a graph prompt reformulates graph data.
    \item \textbf{完善图提示生态。} 我们开发了支持图提示研究的Python库ProG，并构建了用于汇集最新图提示研究成果的综合性网站。
    \item \textbf{指明未来方向。} 我们对当前挑战与未来方向进行了深入探讨。
\end{itemize}
