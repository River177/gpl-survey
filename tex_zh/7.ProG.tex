\section{\texttt{ProG}：图提示的统一库}\label{sec:prog}
加强图提示生态系统的一个不可或缺组成部分是一个精心设计的工具。尽管提出了大量用于通用图学习的工具，但在图提示功能专用库方面仍然存在明显空白。为了弥补这一差距，我们推出了 \texttt{ProG}（Prompt Graph），一个开源的统一库，精心设计以满足图提示的特定需求。这一举措有望通过为研究者和实践者提供多功能且全面的资源，显著提升基于图的应用前景。

%  Description
\texttt{ProG} 是一个基于PyTorch的库，旨在促进预训练GNN的单任务或多任务提示。
架构如图~\ref{fig:prog}所示。
它无缝集成了图提示评估中广泛使用的数据集，包括Cora、CiteSeer、Reddit、
% ~\cite{hamilton2017inductive},
Amazon、
% ~\cite{shiravand2023deep},
和Pubmed等。
% ~\cite{kipf2017semisupervised}.
该工具配备了图提示相关任务中常用的基本评估指标，如准确率、F1分数和AUC分数。值得注意的是，\texttt{ProG} 包含了如 All in One~\cite{sun2023all}、GraphPrompt~\cite{liu2023graphprompt}、GPPT~\cite{sun2022gppt}、GPF~\cite{fang2022prompt} 和 GPF-Plus ~\cite{fang2023universal} 等最先进方法，并且还在持续集成更多图提示模型。总之，\texttt{ProG} 提供以下关键特性：



\begin{itemize}
    \item \textbf{快速启动。} 所有模型都在一致的环境中实现，并配有详细的演示，确保新手能快速入门。
    \item \textbf{完全模块化。} \texttt{ProG} 采用模块化结构，使用户能够根据需要自定义和构建模型。
    \item \textbf{易于扩展。} \texttt{ProG} 设计为无缝扩展以包含更多方法和更广泛的下游任务，适应不断发展的研究需求。
    \item \textbf{标准化评估。} \texttt{ProG} 建立了统一的评估流程，促进跨模型的公平性能比较。
\end{itemize}
