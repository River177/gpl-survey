\section{挑战与未来方向}\label{sec:dis}
\subsection{当前挑战}\label{sec:challenges}
图提示学习近年来取得了显著研究进展，但仍面临若干关键挑战。本小节将对这些挑战进行讨论。

\textbf{图模型的内在限制}
提示（prompt）最早兴起于 NLP 领域 \cite{brown2020language, lester2021power, liu2023pretrain}，其核心目标是释放预训练语言模型在下游任务适配中的潜能。这种“上下文学习（in-context learning）”能力通常在模型参数规模达到一定量级后才会涌现，例如广为人知的大语言模型往往拥有数十亿级别参数。在如此强大的预训练模型支撑下，简单的文本提示即可提炼出适配下游任务的特定知识。相比之下，图任务上的提示学习更为棘手：现有预训练图模型的参数规模通常显著小于 LLMs，使得其下游适配潜力更难充分被“提示”所激发。

\textbf{图提示学习的直观评估}
在 NLP 中，提示通常以离散文本形式出现 \cite{brown2020language, robinson2023leveraging}，因此更便于直观理解、对比与解释。然而，现有图提示多以可学习的 token \cite{sun2022gppt, fang2023universal, fang2022prompt, tan2023virtual, ma2023hetgpt, ge2023enhancing} 或增广图结构 \cite{sun2023all, huang2023prodigy} 的形式呈现，缺少可读的设计表达。这使得图提示难以被直接解释，其有效性往往只能通过下游任务性能间接评估，从而限制了对不同图提示方案进行高效且全面的横向比较。因此，如何构造更直观、可读、可解释的图提示形式仍是一个开放问题。

\textbf{更丰富的下游应用}
当前，图提示学习主要在开源基准上的节点或图分类任务中得到应用 \cite{hamilton2017inductive, xu2018how}。尽管我们在 \secref{sec:app} 中讨论了潜在应用，但其在真实世界中的落地仍然有限。一个具有代表性的案例是将其用于真实交易网络中的欺诈检测，以缓解标签稀缺问题 \cite{wen2023voucher}。然而，与提示技术在 NLP 领域的广泛应用 \cite{bran2023transformers,  robinson2023leveraging, zhang2023benchmarking} 相比，图提示学习在多样真实场景中的应用潜力仍有待深入挖掘。要实现规模化落地，仍需解决两类核心困难：构建强大的、面向领域的预训练图模型，以及为具有独特业务/数据特征的具体应用场景设计合适的提示机制。

\textbf{可迁移的提示设计}
现有图提示学习研究 \cite{liu2023graphprompt, sun2022gppt, gong2023prompt, fang2023universal, fang2022prompt, ge2023enhancing} 多聚焦于在同一数据集上进行预训练与提示调优，因而对跨数据集/跨领域可迁移设计的探索与实证评估仍不充分。尽管 PRODIGY \cite{huang2023prodigy} 探索了同一领域内的迁移性，All in One \cite{sun2023all} 也给出了跨任务与跨领域的实证结果，但总体而言，在更广泛的任务与领域上系统研究提示学习的迁移能力仍较为有限。要实现跨任务、跨领域迁移，需要同时对齐任务空间 \cite{sun2023all}、语义特征 \cite{zhu2023graphcontrol} 与结构模式 \cite{zhao2023graphglow}；这也要求进一步的理论研究为可迁移提示设计提供指导性洞见。

\subsection{未来方向}
基于上述对图提示的分析，我们将未来研究方向总结如下：

\textbf{从类 LLM 的大规模图模型（LGMs）中学习知识。}
目前，图模型往往针对特定领域或任务进行定制，导致其跨领域/跨任务的泛化能力受限 \cite{liu2023graph, zhu2023graphcontrol}。因此，我们期待大规模图模型（Large Graph Models, LGMs）能够成为足够通用的工具，以处理来自不同领域的图任务 \cite{liu2023one, fatemi2023talk}。\citet{liu2023one} 提出的 OFA 是迈向该方向的重要一步，其能够处理来自多领域图数据的分类任务。然而，该方法仍依赖 LLMs 作为通用模板来蒸馏特定领域知识；我们认为，在许多情形下，这一过程有望由合适的图提示技术替代。正如文本提示已被广泛用于将 LLM 适配到各类应用 \cite{robinson2023leveraging, wang2023knowledge, brown2020language}，图提示同样有望从 LGMs 中蒸馏出面向具体下游任务的知识。将图提示应用于 LGMs 可能重塑深度图学习领域，使 LGMs 成为应对多领域、多任务图问题的通用基座。

\textbf{可迁移学习。}
如 \secref{sec:challenges} 所述，现有图提示学习方法 \cite{sun2023all, sun2022gppt, huang2023prodigy, liu2023graphprompt, tan2023virtual, ge2023enhancing, shirkavand2023deep} 多局限于单数据集/单领域设置，缺乏跨任务或跨领域迁移能力。尽管图迁移学习方向已有工作（如 \cite{zhu2023graphcontrol}）尝试实现领域自适应，但专门面向“可迁移图提示”的研究仍较少。为实现迁移能力，提示需要能够对齐不同任务空间 \cite{sun2023all}，将不同图任务重表述为统一模板；同时，还需通过合适的图提示实现结构对齐 \cite{zhao2023graphglow} 与语义对齐 \cite{zhu2023graphcontrol}，以支撑跨领域自适应。未来有望实现真正可迁移的图提示，从而在更广泛场景中扩展图模型的泛化能力与适用范围。

\textbf{更坚实的理论基础。}
尽管图提示学习在多种下游任务上取得了良好效果，但其方法设计大多借鉴 NLP 领域提示调优的经验 \cite{liu2022ptuning, brown2020language, gao2021making, tsimpoukelli2021multimodal, qin2021learning, shin2020autoprompt}。换言之，许多现有图提示方法主要依赖直觉进行构造，其有效性更多通过实证来验证。缺乏充分的理论支撑不仅可能导致性能瓶颈，也会削弱可解释性。因此，从图论与图学习机理出发，为图提示学习建立系统的理论基础，并尽可能缩小理论与实践之间的鸿沟，是一个重要且有潜力的方向。

\textbf{更可解释、更可理解的设计。}
尽管现有图提示学习方法在多种下游任务上展现出显著性能提升 \cite{guo2023datacentric, wen2023voucher, yang2023empirical}，我们仍缺乏对“提示究竟学到了什么”的清晰认识。以提示向量为核心的黑箱学习方式引出了可解释性问题：我们是否能够在输入数据与提示后的图之间建立有意义的对应关系 \cite{sun2023all}？这些问题对于理解与解释提示机制至关重要，但在多数图提示研究中尚未得到充分关注。为了迈向可信的图提示学习，探索自解释能力（self-interpretability）\cite{dai2021selfexplainable}
% , zhang2021protgnn}
以获得对图提示的直观解释，是值得投入的方向。通过理解所学习到的提示向量与结构，我们可以更好地洞察其作用机制并提升可解释性；这将进一步促进图提示在安全或隐私相关下游任务中的有效与可靠应用。
